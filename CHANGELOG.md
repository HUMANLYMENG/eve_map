# 更新日志

## [v1.0.0] - 2026-02-16

### 项目初始化

#### 核心功能
- 数据加载模块，支持 YAML/JSONL 格式
- 星域选择器，支持 113 个星域切换
- Canvas 地图渲染器，支持平移和缩放
- 星系节点渲染（按安全等级着色）
- 星门连线渲染（内部实线，跨域虚线）

#### 交互功能
- 鼠标拖拽平移地图
- 滚轮缩放
- 单击选中星系
- 双击重置视图
- 点击相邻星域星系跳转

#### 视觉设计
- Eveeye 风格深色主题
- 圆角方形节点
- 入口星系绿色发光效果
- 固定大小文字和图标

#### 数据结构
- 使用 EVE SDE 数据（position2D 坐标）
- 支持 5,000+ 个星系
- 相邻星域星系显示

## 技术决策记录

### 坐标系统
- 使用 `position2D` 而非 3D 投影
- Y 轴翻转（北在上）
- 缩放因子 1200/maxRange

### 渲染策略
- 固定大小文字和图标（不随缩放变化）
- 根据缩放级别控制标签密度
- Canvas 分层渲染：连线 -> 外部节点 -> 本域节点 -> 标签

### 性能考虑
- 使用 Map 数据结构存储星系
- 懒加载标签渲染
- 空间索引优化点击检测
