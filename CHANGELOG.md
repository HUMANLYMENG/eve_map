# 更新日志

## [v1.2.0] - 2026-02-18

### 新增功能

#### 坐标系统升级
- 使用 Eveeye 官方坐标数据
- 更准确的星系位置显示

#### 路径记录系统
- 自动记录点击的星系路径
- 显示最近 6 个星系的路径连线
- 路径星系高亮显示（黄色圆圈）
- 不在当前星域的路径星系显示在边界附近
- 多个外部路径星系围绕边界星系均匀分布
- 点击路径记录可快速聚焦到对应星系

#### 路径连线样式
- 有星门连接：橙色实线
- 无星门连接（虫洞）：蓝色虚线
- 连线中点显示方向箭头

#### 虫洞记录系统
- 自动检测路径中的虫洞连接
- 弹出对话框记录虫洞信息：
  - 虫洞类型（100+ 种，支持模糊搜索）
  - 两端信号编号
  - 虫洞大小（S/M/L/XL）
  - 最大寿命（1h/4h/1d/2d）
- 虫洞表格显示：
  - 起点↔终点
  - 类型
  - 大小
  - 记录时间
  - 剩余时间（实时倒计时）
  - 书签名（点击复制到剪贴板）
- 自动删除过期记录

### UI 优化
- 右侧面板宽度增加至 320px
- 优化表格显示，防止文字截断

## 技术决策记录

### 坐标系统
- 使用 `position2D` 而非 3D 投影
- Y 轴翻转（北在上）
- 缩放因子 1200/maxRange

### 渲染策略
- 固定大小文字和图标（不随缩放变化）
- 根据缩放级别控制标签密度
- Canvas 分层渲染：连线 -> 外部节点 -> 本域节点 -> 标签

### 性能考虑
- 使用 Map 数据结构存储星系
- 懒加载标签渲染
- 空间索引优化点击检测
